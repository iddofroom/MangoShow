# 🏗️ ארכיטקטורה - מנגו דרור

## סקירה כללית
אפליקציית Next.js עם עיבוד בצד הלקוח בלבד.

## Stack טכנולוגי
- Next.js 14 + TypeScript
- Recharts - ויזואליזציה
- PapaParse - עיבוד CSV
- Netlify - הוסטינג

## מבנה
\`\`\`
pages/
├── index.tsx         - דף ראשי + דשבורד
├── _app.tsx          - React wrapper
├── _document.tsx     - RTL support
└── 404.tsx           - Error page

utils/
└── dataProcessor.ts  - לוגיקת עיבוד נתונים
\`\`\`

## למידת מחירים
1. איסוף מחירים מהזמנות יחידות
2. חישוב ממוצע לכל מוצר+מיקום+תאריך
3. שימוש במחירים להזמנות מורכבות

## אבטחה
- עיבוד בצד הלקוח בלבד
- אין שמירת נתונים
- HTTPS אוטומטי
